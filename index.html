Daily WOD

The API to get your WOD data.

Resource Description
GET crossfitwicked/wods Returns data for all WODs. Each WOD returned will contain a date and description of the WOD for that day.
GET crossfitwicked/wods/:id Returns a single WOD for the id parameter specified. The WOD returned will contain a date and description of the WOD for that day.
